Angular Alert Messenger

trigger and display bootstrap alerts with the angular framework

Code on Github »

How to use?

The Alert Messenger uses angulars $emit or $broadcast.

To use this Module you have to load "angular-alert-messenger.js" in your index.html and declare it as a dependency of your angular application, like: 

angular.module('your.application',
  ['alert.messenger']);

Then place <alerts></alerts> wherever you want the messages to be shown.

The following code is used in the example on this page. 

$scope.onButtonClick = function () {
  this.$emit("message", {
    'type': $scope.message.type,
    'closeable': $scope.message.closeable,
    'text': $scope.message.text,
    'expires': $scope.message.expires
  });
};
{
  'text': '{{message.text}}',
  'closeable': {{message.closeable}},
  'type': '{{message.type}}',
  'expires': {{message.expires}}
}
